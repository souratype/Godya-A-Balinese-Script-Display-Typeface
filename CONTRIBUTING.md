# Contributing Policy

Thank you for your interest in the Godya typeface project.

## Contribution Status

This project is **open-source but curated**.

- External contributions are **not accepted**
- Pull requests will not be reviewed or merged
- Issues may be closed without action

## Rationale

Godya is a culturally sensitive script typeface developed under a specific design and research framework.  
To maintain authorship clarity, visual consistency, and responsible interpretation of Balinese script, all development decisions are handled solely by the original designer.

## Forking

You are allowed to:
- Fork this repository
- Modify the font for personal or internal use
- Redistribute modified versions under a different name, in accordance with the SIL Open Font License 1.1

Please do **not** submit modifications back to this repository.

Thank you for respecting this policy.
